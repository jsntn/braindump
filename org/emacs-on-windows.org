* Formatter
Right now, I'm using Steve Purcell's [[https://github.com/purcell/emacs-reformatter][reformatter]] with [[https://prettier.io][Prettier]] and [[https://github.com/mvdan/sh][shfmt]].
- Prettier: YAML, CSS/Less/SCSS
- shfmt: Shell script

2. Globally install Prettier using =npm install -g prettier=
